缓动曲线

为了使动画效果更有趣，符合真实的物体运动效果。这个时候缓动曲线函数就派上用场了，GreenSock也提供了各种的运动曲线。

如果使用的是TweenMax的话，它已经包含了EasePack。

EasePack包含下面的这些运动曲线：

1、Back 2、SlowMo 3、StppedEase 4、RoughEase 5、Bounce 6、Circ 7、Elastic 8、Expo 9、Sine

具体各种运动曲线的效果可以去这个地址看。

引入TweenMax.min.js文件

import TeweenMax from '../static/TweenMax.min.js'

然后在项目中使用，vue中实现数字状态过渡。https://cn.vuejs.org/v2/guide/transitioning-state.html data中：

tweenedNumberReportTotal: 0,

mounted中：

TweenLite.to(this.$data, 2, { tweenedNumberReportTotal: reportTotal });
