在vue中引入外部css文件

在vue中引入外部的css文件

在项目的src文件下，新建一个style文件夹，存放css文件。

1. 全局引入

将外部的css文件放到style文件下，引入外部文件只需在main.js文件中

import './style/reset.css'

我引入的是清除默认样式的css文件

2. 局部引入
<style scoped>
  @import '../assets/iconfont/iconfont.css';这个分号一定要写，要不会报错
</style>
